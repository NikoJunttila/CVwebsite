How this gym tracking app works?

1. Create account/login

2. Select a workout program or create one yourself

You find create workout button at the bottom of your profile once logged in HERE Also you see all the workouts you have done.

3. Once you have selected a workout program and logged in you can choose to copy it

4. Once copied you can freely edit it and when you complete a workout and save it the reps and weights will also update making it easy to follow progress

5. When you have done all the exercises in a workout you can save the workout.

6. You can review your workouts and see how much you have trained during specific span of time.
